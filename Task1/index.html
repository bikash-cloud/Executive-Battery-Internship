Adaptive Multi-Cell Battery Intelligence Engine

A production-grade ESP32 based Battery Management System simulation for monitoring a 4-cell lithium battery pack in real time.

View Wokwi Simulation View Source Code

Project Overview

This project monitors four simulated lithium battery cells using ESP32 ADC inputs. It calculates individual cell voltages, pack voltage, average voltage, imbalance percentage, strongest cell, weakest cell, SOC, active faults, and battery health status.
